# Ricky And Morty App

This is a demo app built using [Next.js](https://nextjs.org/) and [Rick And Morty API](https://rickandmortyapi.com/).

## Getting Started

### [Hosted Version](https://rick-morty-gamma.vercel.app/)

### Development Commands

```bash
# Lint
npm run lint

# Typecheck
npm run typecheck

# Test
npm run test

# Dev server
npm run watch
```

### Production Build

```bash
# First run build
npm run build

# run prod server
npm run start
```
